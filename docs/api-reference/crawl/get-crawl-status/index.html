GET /v1/crawl/{id}

Hosted: https://api.fastcrw.com · Self-hosted: http://localhost:3000

Auth (hosted only): Authorization: Bearer $CRW_API_KEY

Self-hosted: no auth required by default.

After POST /v1/crawl returns a job id, poll this endpoint until status == "completed" (or "failed"). 2–5 second intervals are typical.

Path parameter

id — crawl job id returned by POST /v1/crawl (string).

Request

GET /v1/crawl/9f7a...e3b HTTP/1.1
Host: api.fastcrw.com
Authorization: Bearer $CRW_API_KEY

No body. No query parameters.

Response shape

{
  "status": "scraping" | "completed" | "failed",
  "completed": 12,
  "total": 50,
  "creditsUsed": 12,
  "expiresAt": "2026-05-28T10:00:00Z",
  "data": [
    {
      "markdown": "# Page title\n\n...",
      "metadata": {
        "title": "...",
        "sourceURL": "https://example.com/page",
        "statusCode": 200
      }
    }
  ]
}

404 is returned for unknown or expired job ids.

Python (stdlib) polling loop

#!/usr/bin/env python3
import json, os, time, urllib.request

job_id = os.environ["CRW_JOB_ID"]
headers = {"Authorization": f"Bearer {os.environ['CRW_API_KEY']}"}

while True:
    req = urllib.request.Request(
        f"https://api.fastcrw.com/v1/crawl/{job_id}", headers=headers)
    s = json.loads(urllib.request.urlopen(req).read())
    print(s["status"], s.get("completed"), "/", s.get("total"))
    if s["status"] in ("completed", "failed"):
        break
    time.sleep(2)

print("pages:", len(s.get("data", [])))
# Run with: CRW_JOB_ID=... python3 get-crawl-status.py
