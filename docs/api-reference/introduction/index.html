fastCRW API reference

Hosted: https://api.fastcrw.com · Self-hosted: http://localhost:3000

Auth (hosted only): Authorization: Bearer $CRW_API_KEY

Self-hosted: no auth required by default.

fastCRW exposes five Firecrawl-compatible endpoints under /v1. All bodies are JSON; all responses set Content-Type: application/json and wrap successful payloads in {"success": true, ...}.

Endpoints

  • POST /v1/scrape — convert one URL to markdown, html, rawHtml, or JSON. Synchronous.
  • POST /v1/crawl — start an async multi-page crawl; poll GET /v1/crawl/{id} for results.
  • POST /v1/map — discover URLs on a site without scraping page content.
  • POST /v1/extract — structured JSON extraction with a JSON Schema (alias for scrape + formats:[json]).
  • POST /v1/search — web search with optional per-result scraping.

Smoke test

curl -X POST https://api.fastcrw.com/v1/scrape \
  -H "Authorization: Bearer $CRW_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{"url":"https://example.com","formats":["markdown"]}'

Python (stdlib)

#!/usr/bin/env python3
import json, os, urllib.request

req = urllib.request.Request(
    "https://api.fastcrw.com/v1/scrape",
    data=json.dumps({"url": "https://example.com", "formats": ["markdown"]}).encode(),
    headers={"Content-Type": "application/json",
             "Authorization": f"Bearer {os.environ['CRW_API_KEY']}"},
    method="POST",
)
with urllib.request.urlopen(req) as r:
    body = json.loads(r.read())
print(body["success"], list(body["data"].keys()))
# Run with: python3 introduction.py

Error responses

Errors share a common shape:

{"success": false, "error": "message"}

HTTP codes:

  • 400 — bad request (invalid JSON or missing required field)
  • 401 — unauthorized (hosted only; missing or invalid Bearer token)
  • 404 — not found (e.g. unknown crawl job id)
  • 429 — rate limited (back off and retry)
  • 503 — search backend disabled or unreachable
