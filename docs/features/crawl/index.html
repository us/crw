POST /v1/crawl

Hosted: https://api.fastcrw.com · Self-hosted: http://localhost:3000

Auth (hosted only): Authorization: Bearer $CRW_API_KEY

Self-hosted: no auth required by default.

Crawl is async: POST /v1/crawl returns a job id; poll GET /v1/crawl/{id} until status == "completed".

Request body

{
  "url": "https://example.com",
  "limit": 50,
  "maxDepth": 3,
  "includePaths": ["^/docs/.*"],
  "excludePaths": ["^/blog/.*"],
  "scrapeOptions": { "formats": ["markdown"], "onlyMainContent": true }
}

Required: url. Optional: limit (default 10), maxDepth, includePaths/excludePaths regex arrays, scrapeOptions (forwarded to each page scrape).

Response shape (job accepted)

{
  "success": true,
  "id": "9f7a...e3b",
  "url": "https://api.fastcrw.com/v1/crawl/9f7a...e3b"
}

Polling shape

{
  "status": "scraping" | "completed" | "failed",
  "completed": 12,
  "total": 50,
  "data": [ { "markdown": "...", "metadata": { "sourceURL": "..." } } ]
}

Python (stdlib)

#!/usr/bin/env python3
import json, os, time, urllib.request

def post(path, body):
    req = urllib.request.Request(
        f"https://api.fastcrw.com{path}",
        data=json.dumps(body).encode(),
        headers={"Content-Type": "application/json",
                 "Authorization": f"Bearer {os.environ['CRW_API_KEY']}"},
        method="POST")
    return json.loads(urllib.request.urlopen(req).read())

job = post("/v1/crawl", {"url": "https://example.com", "limit": 20})
job_id = job["id"]

while True:
    req = urllib.request.Request(
        f"https://api.fastcrw.com/v1/crawl/{job_id}",
        headers={"Authorization": f"Bearer {os.environ['CRW_API_KEY']}"})
    s = json.loads(urllib.request.urlopen(req).read())
    if s["status"] in ("completed", "failed"):
        break
    time.sleep(2)

print(len(s["data"]), "pages")
# Run with: python3 crawl.py
