POST /v1/extract

Hosted: https://api.fastcrw.com · Self-hosted: http://localhost:3000

Auth (hosted only): Authorization: Bearer $CRW_API_KEY

Self-hosted: no auth required by default.

Extract is an ergonomic alias for /v1/scrape with formats:["json"] plus a JSON schema. Use it to pull typed fields (title, price, author, etc.) from one or more URLs.

Request body

{
  "urls": ["https://example.com/product/1"],
  "prompt": "Return product name and price",
  "schema": {
    "type": "object",
    "properties": {
      "name": { "type": "string" },
      "price": { "type": "number" }
    },
    "required": ["name", "price"]
  }
}

Required: urls array (1+). Optional: prompt string, schema JSON Schema object, systemPrompt.

Response shape

{
  "success": true,
  "data": {
    "name": "Widget Pro",
    "price": 19.99
  }
}

Single URL: data matches the schema directly. Multiple URLs: data is an array keyed by URL order.

Python (stdlib)

#!/usr/bin/env python3
import json, os, urllib.request

payload = {
    "urls": ["https://example.com/product/1"],
    "schema": {
        "type": "object",
        "properties": {"name": {"type": "string"}, "price": {"type": "number"}},
        "required": ["name", "price"],
    },
}
req = urllib.request.Request(
    "https://api.fastcrw.com/v1/extract",
    data=json.dumps(payload).encode(),
    headers={"Content-Type": "application/json",
             "Authorization": f"Bearer {os.environ['CRW_API_KEY']}"},
    method="POST",
)
with urllib.request.urlopen(req) as r:
    body = json.loads(r.read())

print(body["data"])
# Run with: python3 extract.py
