POST /v1/map

Hosted: https://api.fastcrw.com · Self-hosted: http://localhost:3000

Auth (hosted only): Authorization: Bearer $CRW_API_KEY

Self-hosted: no auth required by default.

Map discovers URLs reachable from a seed without scraping page content. Use it to plan a crawl, build a sitemap, or filter links before a scrape pass.

Request body

{
  "url": "https://example.com",
  "search": "docs",
  "limit": 100,
  "includeSubdomains": false
}

Required: url. Optional: search substring filter, limit (default 5000), includeSubdomains boolean, ignoreSitemap boolean.

Response shape

{
  "success": true,
  "links": [
    "https://example.com/",
    "https://example.com/docs",
    "https://example.com/docs/quickstart"
  ]
}

Top-level links is a flat string array (not objects). Order is discovery order; dedup is applied.

Python (stdlib)

#!/usr/bin/env python3
import json, os, urllib.request

req = urllib.request.Request(
    "https://api.fastcrw.com/v1/map",
    data=json.dumps({"url": "https://example.com", "limit": 50}).encode(),
    headers={
        "Content-Type": "application/json",
        "Authorization": f"Bearer {os.environ['CRW_API_KEY']}",
    },
    method="POST",
)
with urllib.request.urlopen(req) as r:
    body = json.loads(r.read())

for link in body["links"][:20]:
    print(link)
# Run with: python3 map.py
