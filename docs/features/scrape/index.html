POST /v1/scrape

Hosted: https://api.fastcrw.com · Self-hosted: http://localhost:3000

Auth (hosted only): Authorization: Bearer $CRW_API_KEY

Self-hosted: no auth required by default.

Request body

{
  "url": "https://example.com",
  "formats": ["markdown"],
  "onlyMainContent": true,
  "waitFor": 0,
  "timeout": 30000
}

Required: url. Optional: formats array of markdown | html | rawHtml | json | links | screenshot; onlyMainContent boolean; waitFor ms; timeout ms; jsonOptions object (when formats includes json).

Response shape

{
  "success": true,
  "data": {
    "markdown": "# Heading\n\n...",
    "html": "<h1>Heading</h1>...",
    "metadata": {
      "title": "...",
      "description": "...",
      "sourceURL": "https://example.com",
      "statusCode": 200
    }
  }
}

Top-level data is an object (not array). Format keys (markdown, html, rawHtml, json) appear only when requested in formats.

Python (stdlib)

#!/usr/bin/env python3
import json, os, urllib.request

req = urllib.request.Request(
    "https://api.fastcrw.com/v1/scrape",
    data=json.dumps({"url": "https://example.com", "formats": ["markdown"]}).encode(),
    headers={
        "Content-Type": "application/json",
        "Authorization": f"Bearer {os.environ['CRW_API_KEY']}",
    },
    method="POST",
)
with urllib.request.urlopen(req) as r:
    body = json.loads(r.read())

print(body["data"]["markdown"][:500])
# Run with: python3 scrape.py
