POST /v1/search

Hosted: https://api.fastcrw.com · Self-hosted: http://localhost:3000

Auth (hosted only): Authorization: Bearer $CRW_API_KEY

Self-hosted: no auth required by default.

Search returns ranked web results. Pass scrapeOptions to scrape each result in the same call (saves a round trip per result).

Request body

{
  "query": "rust async runtime",
  "limit": 5,
  "scrapeOptions": { "formats": ["markdown"], "onlyMainContent": true }
}

Required: query. Optional: limit (default 5), scrapeOptions object (same shape as POST /v1/scrape body minus url).

Response shape

{
  "success": true,
  "data": [
    {
      "url": "https://...",
      "title": "...",
      "snippet": "LLM-ready summary line",
      "description": "Same as snippet (SearXNG-native name)",
      "position": 1,
      "score": 9.5,
      "category": "general"
    }
  ]
}

Results nest under data array. snippet and description hold the same value (Firecrawl-compatible + native).

Python (stdlib)

#!/usr/bin/env python3
import json, os, urllib.request

req = urllib.request.Request(
    "https://api.fastcrw.com/v1/search",
    data=json.dumps({"query": "rust async runtime", "limit": 3}).encode(),
    headers={"Content-Type": "application/json",
             "Authorization": f"Bearer {os.environ['CRW_API_KEY']}"},
    method="POST",
)
with urllib.request.urlopen(req) as r:
    body = json.loads(r.read())

for row in body["data"]:
    print(row["position"], row["title"], row["url"])
# Run with: python3 search.py
