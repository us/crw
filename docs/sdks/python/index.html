fastCRW Python SDK examples

Hosted: https://api.fastcrw.com · Self-hosted: http://localhost:3000

Auth (hosted only): Authorization: Bearer $CRW_API_KEY

Self-hosted: no auth required by default.

fastCRW has no proprietary client. The REST API is Firecrawl-compatible, so you can either point the official firecrawl-py client at api.fastcrw.com, or use plain requests.

Install: pip install requests

1. Scrape one URL

import os, requests
r = requests.post("https://api.fastcrw.com/v1/scrape",
    json={"url": "https://example.com", "formats": ["markdown"]},
    headers={"Authorization": f"Bearer {os.environ['CRW_API_KEY']}"})
print(r.json()["data"]["markdown"][:500])
# Run with: python3 scrape_example.py

2. Crawl a site (async)

import os, time, requests
h = {"Authorization": f"Bearer {os.environ['CRW_API_KEY']}"}
job = requests.post("https://api.fastcrw.com/v1/crawl",
    json={"url": "https://example.com", "limit": 20}, headers=h).json()
while True:
    s = requests.get(f"https://api.fastcrw.com/v1/crawl/{job['id']}", headers=h).json()
    if s["status"] in ("completed", "failed"): break
    time.sleep(2)
print(len(s["data"]), "pages")
# Run with: python3 crawl_example.py

3. Map URLs on a site

import os, requests
r = requests.post("https://api.fastcrw.com/v1/map",
    json={"url": "https://example.com", "limit": 50},
    headers={"Authorization": f"Bearer {os.environ['CRW_API_KEY']}"})
for u in r.json()["links"][:10]: print(u)
# Run with: python3 map_example.py

4. Extract structured JSON

import os, requests
r = requests.post("https://api.fastcrw.com/v1/extract",
    json={
        "urls": ["https://example.com/product/1"],
        "schema": {"type": "object",
                   "properties": {"name": {"type": "string"},
                                  "price": {"type": "number"}},
                   "required": ["name", "price"]},
    },
    headers={"Authorization": f"Bearer {os.environ['CRW_API_KEY']}"})
print(r.json()["data"])
# Run with: python3 extract_example.py

5. Search the web

import os, requests
r = requests.post("https://api.fastcrw.com/v1/search",
    json={"query": "rust async runtime", "limit": 3},
    headers={"Authorization": f"Bearer {os.environ['CRW_API_KEY']}"})
for row in r.json()["data"]:
    print(row["position"], row["title"], row["url"])
# Run with: python3 search_example.py
